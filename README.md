# credit_risk_engine_4966
Motor de risco de crédito em Python, alinhado à Resolução CMN 4.966/21 e ao framework IRB. O sistema estima PD, LGD e EAD, calcula ECL de 12 meses e lifetime, incorpora cenários macroeconômicos e evolui para uma camada de gestão de portfólio e um índice de atratividade ajustado ao risco para apoiar decisões de originação e precificação.
